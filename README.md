# Muraafiq Plus

*Platform:* Manus (No-Code/Low-Code)  
*Description:* Muraafiq+ is a smart companion service for people with disabilities, allowing them to book verified assistants via a digital platform.

## Prototype
The MVP was developed on Manus. You can try it using this link: [Launch Muraafiq+ Prototype]([YOUR_MANUS_LINK_HERE](https://morafiqplus-e8berhgd.manus.space/))

## Features
- Digital booking of companions
- AI-powered matching for safe and efficient assistance
- Real-time tracking and notifications
- Emergency alert system
- User feedback and rating system

## Future Improvements
- Full app development using Python for AI backend
- Mobile app using Flutter for iOS and Android
- Advanced AI matching and analytics
